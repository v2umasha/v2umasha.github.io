Projects

IBERT — Inference BERT Accelerator GitHub ↗

FPGA-based hardware accelerator for BERT inference. Designed a systolic array with FSM-based address generation controllers for multi-bank SRAMs, and a two-FSM attention head with input mux and output demux. Implemented fixed-point exp(x) and softmax pipelines using argument-reduction and Horner approximation. RTL-to-GDS on ASAP7 targeting 1 GHz; critical path analysis via OpenROAD with post-route timing closure achieving 0.739 ns slack.

SystemVerilog ASAP7 OpenROAD FPGA Fixed-Point Systolic Array
RAIHA — Robotic AI Hardware Accelerator GitHub ↗

RTL implementation of an activation preprocessing pipeline and AXI host interface targeting OpenVLA-7B inference acceleration (LLaMA-2 backbone), aiming to push throughput from ~5 Hz to 25–50 Hz for robotic manipulation. Modules include a reduction tree, scale register file, BF16→FP32 multiplier via DSP48E2, and a downstream divider pipeline. 224 golden truth files captured via PyTorch forward hooks for RTL comparison. Quantization research: per-group W4A4 (gs=64) achieved 88.4% weighted task accuracy vs. 83.1% BF16 baseline.

SystemVerilog AXI BF16 DSP48E2 PyTorch INT4 Quantization

Before — Per-tensor W4A4 (0% task success)

After — Per-group gs=64 (88.4% task success)

5-Stage Pipelined RISC-V SoC

Pipelined RISC-V CPU synthesized and deployed on a PYNQ-Z1 FPGA, extended to a full SoC with memory-mapped PWM and SPI peripherals. Two-FF synchronizer on the byte-ready signal crossing the SCLK domain to the system clock domain. SPI master writes to shared PWM config registers with a CPU-wins priority mux.

SystemVerilog RISC-V FPGA SPI PWM CDC
256-Bit Scan Chain — Full-Custom IC (TSMC 65nm)

Designed and laid out a 256-bit scan chain using Transmission Gate Flip-Flops (TGFF) in Cadence Virtuoso on TSMC 65nm. 20 transistors per cell vs. 32 for a standard NAND-based FF — a 37% reduction enabled by the contention-free topology. H-tree clock distribution, Monte Carlo analysis, pre/post-layout simulation, with power-delay product as the figure of merit.

Cadence Virtuoso TSMC 65nm Full-Custom DFT Monte Carlo

MXA — OCP MX GEMM Accelerator GitHub ↗

Full design of an OCP Microscaling (MXFP4) GEMM accelerator subsystem targeting 1 GHz post-route on ASAP7 and 512 GOPS peak throughput. Architecture spans a pipelined FP4 dot-product datapath, 16×16 PE array with cocotb DV, banked SRAM memory subsystem, burst-capable AXI4 Full DMA master, and full-system integration with CDC. Formally verified via SymbiYosys; GitHub Actions CI gating each phase.

SystemVerilog MXFP4 AXI4 cocotb SymbiYosys ASAP7 CDC
QoS AXI Interconnect Fabric GitHub ↗

Multi-domain AXI interconnect with quality-of-service arbitration. Submodules include a selective dispatch queue, AXI4-Stream wrapper, QoS arbiter, address decoder/crossbar, CDC crossing, APB peripheral bridge, and clock gating/power domain analysis. Architecture crosses a fast CPU/fabric domain to a slow APB peripheral domain. UPF power domain definitions, SVA assertions, and quantitative timing closure targets via OpenROAD/ASAP7.

SystemVerilog AXI4-Stream APB CDC UPF SVA OpenROAD
